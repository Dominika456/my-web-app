# Магазин цветов «Лаванда»

Интернет-магазин цветов с доставкой. Разработан на Node.js + Express + PostgreSQL.

## Функциональность

- Авторизация и регистрация пользователей
- Админ-панель для управления товарами (CRUD)
- Поиск и фильтрация товаров
- Пагинация
- Экспорт товаров в Excel

## Технологии

- Node.js
- Express
- PostgreSQL
- bcrypt (хеширование паролей)
- express-session (сессии)
- ExcelJS (экспорт в Excel)

## Установка и запуск

1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Создать файл `.env` с переменными окружения
4. Запустить сервер: `node server.js`
